# 🤖 PromptCraft AI

> **An AI-powered Prompt Optimizer built using Python, Streamlit, and Google Gemini API.**

PromptCraft AI helps users transform simple prompts into well-structured, professional prompts that generate higher-quality AI responses. It also analyzes prompt quality and provides suggestions for improvement.

# 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/nazsayyad112/PromptCraft-AI.git
```

### Navigate into the project

```bash
cd PromptCraft-AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the Application

```bash
streamlit run app.py
```

# 🎯 Key Features Explained

### 🔹 Prompt Analysis
Analyzes the prompt quality and generates a score with improvement suggestions.

### 🔹 Prompt Optimization
Uses Google Gemini to rewrite prompts into professional, structured prompts.

### 🔹 Prompt History
Stores prompts during the current session using Streamlit Session State.

### 🔹 Download Feature
Allows users to save optimized prompts as text files.
