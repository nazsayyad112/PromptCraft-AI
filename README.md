# 🤖 PromptCraft AI

An AI-powered Prompt Optimizer built using **Python**, **Streamlit**, and the **Google Gemini API**. This application helps users transform simple prompts into well-structured, professional prompts that generate better AI responses.

---

## ✨ Features

- 🤖 AI-powered prompt optimization using Google Gemini
- 📊 Prompt quality analysis and scoring
- 💡 Suggestions to improve prompts
- 📂 Category-based prompt optimization
- 🎭 Tone selection
- 📏 Output length selection
- 📜 Prompt history
- 📥 Download optimized prompts
- 🎨 Clean Streamlit user interface

 🚀 Installation

 1. Clone the repository

```bash
git clone https://github.com/nazsayyad112/PromptCraft-AI.git
```

 2. Navigate into the project

```bash
cd PromptCraft-AI
```

 3. Create a virtual environment

```bash
python -m venv venv
```

 4. Activate the virtual environment

Windows:

```bash
venv\Scripts\activate
```

 5. Install dependencies

```bash
pip install -r requirements.txt
```

 6. Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

 7. Run the application

```bash
streamlit run app.py
```
